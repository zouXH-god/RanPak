## 问题需求

ssh工具，连接信息中加入 备注 项，可以为该连接加入备注信息，并将备注也纳入检索，当连接的host为域名而不是ip时，自动解析获取其对应ip加入备注信息中
新增密钥管理，可以选择文件创建也可以直接输入文本，密钥由应用进行保存，不使用本地选择的路径地址
配置导入也支持导入密钥的格式
 {"data":[{"ip":"127.0.0.1","port":22,"username":"root","private":"pre","name":"本地测试机"}], "privates": [{"name": "pre", "private": "**********", "password": "123456"}]}

## 项目约束

- 代码需遵循 Clean Code 原则，对复杂逻辑进行拆分封装，对不同的模块按照功能文件夹存放，代码关键部分使用中文注释解释，每个函数需要有注释说明
- 后端采用 uv 进行管理
